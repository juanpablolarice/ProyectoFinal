1) Clonar repositorio
```git clone https://github.com/juanpablolarice/ProyectoFinal.git```

```npm install```
3) Configurar cuenta de firestore en index.js
```const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: ""
};```
4) Iniciar el servidor 
```npm start```
